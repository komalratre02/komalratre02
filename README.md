### 🚀 Highlight Projects

#### [Distributed Rate Limiter](https://github.com/komalratre02/distributed-rate-limiter)
A production-grade, distributed rate limiting engine built with **Java 21**, **Spring WebFlux**, **Redis Lua Scripts**, and **Apache Kafka**.
- Atomic distributed coordination via Redis Sorted Sets & Lua
- Graceful degradation circuit breakers without bulky libraries
- O(1) Token Bucket and Sliding Window implementations
- Fire-and-forget Kafka telemetry

Conceptual multi-agent system utilizing **LangGraph** orchestration to model Planner, Research, Writer, and Reviewer workflows.
- Live **RAG pipeline** with semantic retrieval via Vector DBs
- Autonomous self-correction workflows based on eval metrics
- Resilient Model Gateway for seamless LLM provider failover
